# IPMAT Launchpad — LPT Edtech

A modern landing page for LPT Edtech's IPMAT coaching program. Built with React, TypeScript, Vite, and Tailwind CSS.

## Features

- Responsive design optimized for all devices
- Lead capture form with backend integration
- Conversion-optimized UI for Meta Ads traffic
- Testimonials, FAQ, results, and center information sections

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
